# Online Calculator

Uma calculadora moderna e responsiva rodando direto no navegador, com suporte a teclado e operações encadeadas.

🧮 **[Acesse o projeto ao vivo](https://daviv22.github.io/calculator)**

## Funcionalidades

- **Operações básicas** — adição, subtração, multiplicação e divisão
- **Operações encadeadas** — realize múltiplos cálculos em sequência sem perder o contexto
- **Repetição com `=`** — pressionar `=` repetidamente repete a última operação
- **Números decimais** — suporte a ponto flutuante
- **Troca de sinal** — botão `+/-` para alternar entre positivo e negativo
- **Deletar dígito** — botão `del` para apagar o último caractere inserido
- **Suporte a teclado** — use os números, operadores, `Enter`, `Backspace` e `Esc`

## Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)

## Como executar localmente

Não há dependências ou build necessário. Clone o repositório e abra o `index.html` no navegador:

```bash
git clone https://github.com/seu-usuario/calculator.git
cd calculator
```

Abra o arquivo `index.html` diretamente no navegador, ou use uma extensão como o **Live Server** no VS Code.

## Atalhos de teclado

| Tecla | Ação |
|---|---|
| `0` – `9` | Inserir dígito |
| `+` `-` `*` `/` | Operadores |
| `Enter` | Calcular resultado |
| `Backspace` | Apagar último dígito |
| `Escape` | Limpar tudo |
| `.` | Inserir ponto decimal |
